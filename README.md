## 📖 如何查看 PDF 笔记

由于 PDF 文件较大（56.6 MB），GitHub 在线预览可能加载较慢或失败。  
**推荐直接下载到本地查看**：

1. 在仓库中找到 `笔记/韩顺平_循序渐进学Java零基础【完整笔记】.pdf` 文件
2. 点击文件名进入详情页
3. 点击右上角的 **`Download`** 按钮即可下载
4. 用本地 PDF 阅读器（如 Adobe Acrobat、浏览器自带阅读器）打开

> 💡 如果希望在线查看，可以尝试使用 [GitHub Pages 链接](https://xu-linghui.github.io/Java-Notes/%E7%AC%94%E8%AE%B0/%E9%9F%A9%E9%A1%BA%E5%B9%B3_%E5%BE%AA%E5%BA%8F%E6%B8%90%E8%BF%9B%E5%AD%A6Java%E9%9B%B6%E5%9F%BA%E7%A1%80%E3%80%90%E5%AE%8C%E6%95%B4%E7%AC%94%E8%AE%B0%E3%80%91.pdf)，但加载速度取决于网络环境。
